# 打不开IDEA？

由于你安装了其它的破解补丁，需要去看看idea64.exe.vmoptions的配置对不对？

建议去掉之前的 -javaagent 的配置。然后，再根据上面的教程安装咱们的破解补丁~

vmoptions找不到？打开IDEA，菜单栏:Help -> Edit Custom Properties，进行修改即可

1.补丁就是不要拖进去之后，就删了，补丁放到固定的位置拖进去，之后不要更换补丁位置 2.之前版本，没有删除干净也不行，一定要删除，卸载干净 3.补丁位置和IDEA 不要装在C 盘

注意：如果提示 key is invalid ,说明agent未激活，找到C:\Users\用户名\AppData\Roaming\JetBrains下面的IntelliJIdea2020.1文件夹删掉（删掉会把所有配置都删除），重启idea

卸载干净注意事项：

删除：“C:\Users\User”下的 “IntelliJIdea”文件夹中的所有文件

比如：我的用户是“root” 那就是这个位置 C:\Users\root\AppData\Local\JetBrains\IntelliJIdea*

删除：c盘 带 IDEA 的文件夹
